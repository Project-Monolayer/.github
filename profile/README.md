# Monolayer

## Introduction
Monolayer is a DeFi-trader centric Wallet as a Service (WaaS) API

## Purpose
Monolayer aims to completely eliminate the technical and financial barrier of entering DeFi trading by providing cost effective data and trading solutions

## Features
- API-managed wallets
- Lowest latency
- High level abstractions for all chains
- Generic syntax across all chains and protocols
- Smart order routers
- Bundling, bridging and cross chain swaps
- Guaranteed finality
